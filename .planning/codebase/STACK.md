## Platform Requirements

**Development:**
- Ruby 3.4.9 (managed via `.ruby-version`)
- Node.js (for `npm run tokens:build` and Biome)
- libvips, libyaml, libpq, libjemalloc2 (system libs for image processing + Postgres + memory)
- Postgres 16 with `pgvector` extension (Docker image: `pgvector/pgvector:pg16`)

**Production:**
- Docker image built from multi-stage `Dockerfile` (Ruby 3.4.9-slim base, non-root `rails` user, listens on `:3000`)
- Pushed image: `ghcr.io/we-promise/sure:stable` (per `compose.example.yml`)
- Optional Helm chart: `charts/sure` (`0.7.3-alpha.2`, `kubeVersion >=1.25.0-0`) — supports CloudNativePG + redis-operator subcharts
- Web + worker split (two containers sharing the same image, `web` runs `rails server`, `worker` runs `bundle exec sidekiq`)
- Backups via `prodrigestivill/postgres-backup-local` (`compose.example.yml`)
- Pipelock sidecar proxy for AI egress filtering (`compose.example.ai.yml`, `ghcr.io/luckypipewrench/pipelock:2.8.0`)
